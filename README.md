# PDF Extractor

PDF Extractor is a web application that allows users to upload PDF files and automatically extracts, analyzes, and reformats their content for easy reading and structured data extraction.

## Features
- **Upload PDFs:** Upload a single PDF file at a time for processing.
- **Automatic Parsing:** Extracts raw text, headings, bullet points, tables, and more from your PDF.
- **Readable Output:** Displays the extracted content in a visually pleasing, notebook-style interface.
- **Structured Data:** Uses AI to identify document structure, headings, lists, tables, and metadata.
- **Results Page:** View the parsed content, metadata, and raw text in a user-friendly format.

## Usage
1. Visit the app at [http://localhost:5173/](http://localhost:5173/) (until cloud hosting is complete).
2. Upload your PDF file using the interface.
3. View the extracted and formatted content on the results page.
